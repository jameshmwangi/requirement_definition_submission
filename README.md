# README

## Tech Stack

- Ruby 3.3.0
- Rails 7.0.10

## Setup Instructions

```bash
$ git clone git@github.com:jameshmwangi/rafiki_house_hunter_app.git
$ cd rafiki_house_hunter_app
$ bundle install
$ rails db:create && rails db:migrate
$ rails s
```
